Day 1 JavaScipt Practice

We already know about variables. A variable can store one value. If we use an object, we can collect many variables into a single "filing cabinet" to organize similar variables.

For example, all the variables that relate to a certain player can be put into a player object. For example...

    let playerOne, playerTwo;
    
    playerOne = {};
    playerOne.name = "Joe";
    playerOne.score = 54;
    
    playerTwo = {};
    playerTwo.name = "Jane";
    playerTwo.score = 65;

There are three things to remember when creating and using objects.

  1. Create the object with curly brackets {}
  2. Assign the properties of the object with dot notation.
  3. Access the properties of the object with dot notation.
